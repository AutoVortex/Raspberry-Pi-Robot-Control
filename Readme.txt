Proiectul nostru se numeste AutoVortex Lego Rasperry Pi Robot Control (LegoRapi) si pe scurt se doreste a fi o solutie practica pentru a
controla un robot construit din piese Lego cu ajutorul unui minicalculator Raspberry Pi.
Grupul tinta este format din copii care doresc o solutie rapida si simpla de conectare. Acestia au nevoie pe langa robot doar de un mini 
calculator Raspberry PI, de o baterie solara si o tastatura wireless.
