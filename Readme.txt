Proiectul nostru se numeste AutoVortex Lego Rasperry Pi Robot Control (LegoRapi) si pe scurt se doreste a fi o solutie practica pentru a controla un robot construit din piese Lego cu ajutorul unui minicalculator Raspberry Pi.
Grupul tinta este format din copii care doresc o solutie rapida si simpla de conectare. Acestia au nevoie pe langa robot doar de un mini calculator Raspberry PI, de o baterie solara si o tastatura wireless sau cu ajutorul unei tablete.
Pentru showoff v-am pregatit o scurta demonstratie a robotului nostru IGOR care a castigat Campionatul Central si Est European de Robotica FIRST Tech Challenge.
Cu acest robot echipa AutoVortex va participa la Campionatul Mondial de Robotica din Saint Louis, USA intre 24-28 aprilie 2013.
